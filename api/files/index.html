Document Summarizer PoC

Este proyecto es una prueba de concepto (PoC) para un resumidor de documentos utilizando la API de OpenAI. El objetivo es demostrar cómo se puede integrar la API para generar resúmenes de texto de manera eficiente y precisa.

Costos AWS Bedrock

El costo de utilizar la API de AWS Bedrock para resúmenes y explicación de documentos depende del modelo que se utilice y la cantidad de tokens procesados. A continuación, se detallan los costos aproximados para diferentes modelos:

Modelo Input (USD x 1M tokens) Output (USD x 1M tokens)
Claude Sonnet 4.6 $3,00 $15,00
Claude Haiku 4.5 $1,00 $5,00
Amazon Nova Pro $0,80 $3,20
Cohere Command R+ $2,50 $10,00

Estimación

Procesando 200 documentos diarios, cada uno de unos 18.000 a 20.000 tokens de entrada. Se asume que cada resumen genera unos 2500 tokens de salida.
